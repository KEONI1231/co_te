# [문제]
## - 알파벳 대문자와 숫자(0~9)로만 구성괸 문자열이 입력으로 주어집니다. 이때 모든 알파벳을 오름차순으로 정렬하여 이어서 출력한 뒤에, 그 뒤에 모든 숫자를 더한 값을 이어서 출력합니다.


## - 예를 들어 K1KA5CB7이라는 값이 들어오면 ABCKK13을 출력합니다.

## - 나이트는 특정 위치에서 다음과 같은 2가지 경우로 이동할 수 있습니다.

### 입력예시1
    K1KA5CB7
### 출력예시1
    ABCKK13

### 출력예시2
    AJKDLSI412K4JSJ9D\
### 출력예시2
    ADDIJJKKLSS20
